Reflektion

Det här arbetet var lite svårt i början eftersom jag inte hade så mycket koll på SQL och hur allting hänger ihop. Jag fick testa mig fram ganska mycket. När jag väl förstod hur man gör tabeller och hur de kopplas med id så började det kännas lättare.

Det som gick bra var att skriva de enklare SQL-kommandona som SELECT och INSERT. Det kändes logiskt. Det svåra var när jag skulle göra JOIN och GROUP BY, det tog ett tag innan jag fattade varför det behövdes och hur man skrev det rätt.

LINQ-delen var också lite klurig för jag är inte van vid att skriva C# på det sättet. Men efter att jag jämförde med SQL så började jag förstå hur LINQ fungerar.

Säkerhetsdelen var ny för mig, speciellt SQL injection. Nu vet jag i alla fall varför man inte ska stoppa in text direkt i frågor.

Om jag skulle göra om uppgiften skulle jag planera lite bättre och försöka förstå helheten från början. Jag tycker ändå att jag lärde mig mycket och nu känner jag mig lite tryggare med databaser än innan.
